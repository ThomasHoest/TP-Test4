# Goals — what you're aiming at

What you're working toward, over the next 12 months / 90 / 30 days. The assistant
uses these to weight what to surface and what to set aside.

## What's here

- `goals.md` — your goals, kept as sections.

## When to write here vs. elsewhere

- A target or aim → here.
- A concrete, multi-step effort toward a goal → a project in [`../projects/`](../projects/).
- A single to-do → a task in [`../tasks/`](../tasks/).
