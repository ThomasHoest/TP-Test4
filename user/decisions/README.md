# Decisions — what you and the assistant decided

A single running log (`decisions.md`) of decisions made between you and your
assistant, kept somewhere easy to reach so you can scan what's been settled.
Append-only; each entry is dated.

## When to write here vs. elsewhere

- An actual decision reached → append a dated entry to `decisions.md`.
- A larger effort the decision is part of → a project in [`../projects/`](../projects/).
- A follow-up to do → a task in [`../tasks/`](../tasks/).
